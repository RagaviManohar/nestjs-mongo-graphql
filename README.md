# nestjs-mongodb-graphql - Backend

This repo contains all the backend related stuffs for nestjs-mongodb-graphql

## Description

We have used [Nest](https://github.com/nestjs/nest) for our nestjs-mongodb-graphql backend.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app

```bash
# controller
$ nest generate controller user

# service
$ nest generate service user

# resolver
$ nest generate resolver user

# module
$ nest generate module user
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
